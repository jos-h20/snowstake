Privacy Policy

Effective Date: December 2025

The "Mt. Bachelor Snow Stake" Chrome extension is designed to be a simple, privacy-first tool for checking snow conditions.

1. Data Collection

This extension does not collect, store, or transmit any personal data, browsing history, or user metrics. It is a "stateless" tool that only displays a publicly available live stream.

2. Third-Party Services

This extension embeds a YouTube live stream. When you view the stream, YouTube may collect data according to their own privacy policy. We do not have access to this information.

3. Permissions

The extension requires minimal permissions solely to allow the YouTube video to bypass security headers and display correctly within the popup window.

4. Contact

If you have questions, you can contact the developer via the GitHub repository where this project is hosted.
